# Adding your links

All project and research links live in this folder. Edit the files and replace placeholders with your real URLs.

## Projects (`projects.ts`)

- **githubUrl** — Your GitHub repo (e.g. `https://github.com/yourusername/repo-name`).
- **demoUrl** — Optional. Live demo / running example (e.g. Vercel, Netlify, or any public URL). Remove the field or leave empty if you don’t have a demo.

Cards show a GitHub icon and a “live demo” icon when these URLs are set.

## Research / papers (`research.ts`)

- **pdfUrl** — Link to the paper (e.g. arXiv, your site, or a direct PDF URL).
- **codeUrl** — Optional. Link to the code (e.g. GitHub repo).

Cards show **PDF** and **Code** links when these are set.

## Quick checklist

1. **Projects:** In `projects.ts`, replace `yourusername` in each `githubUrl` with your GitHub username and repo name. Add `demoUrl` only for projects that have a live demo.
2. **Research:** In `research.ts`, replace the `pdfUrl` and `codeUrl` placeholders with your paper and repo links.
